OONI:

Open Observatory of Network Interference

A global observation network for detecting censorship, surveillance and traffic manipulation on the internet.

Anyone can run a probe to test their own connection and help us better understand the topology of network surveillance. OONI is free software and the measurements it collects are released under an open license.

Run an OONI Probe

How OONI Works

OONI scans TCP, DNS, HTTP and TLS connections for tampering. Some tests work by requesting data over an unencrypted connection and comparing against a known good value. Others check for DNS spoofing, keyword filtering, transparent proxying and website block lists.
